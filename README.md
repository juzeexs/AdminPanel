# 🚀 LiteAdmin - Dashboard Profissional

**LiteAdmin** é um painel administrativo moderno, performático e totalmente responsivo. Focado em uma experiência de usuário (UX) fluida, o projeto apresenta uma interface limpa com suporte a **Dark Mode**, gráficos interativos e gerenciamento dinâmico de dados.

📍 **Live Demo:** [adminreact-kappa.vercel.app](https://adminreact-kappa.vercel.app)

---

## ✨ Destaques do Projeto

O layout foi projetado utilizando a fonte *Plus Jakarta Sans* e ícones *Material Symbols*, garantindo um visual contemporâneo e profissional.

* **Dashboard Geral:** Visão rápida de KPIs (Receita, Pedidos, Clientes e Cancelamentos).
* **Gráficos Interativos:** Visualização de vendas, lucro, categorias e metas utilizando a biblioteca Chart.js.
* **Modo Escuro (Dark Mode):** Alternância dinâmica de temas com ajuste de paleta de cores em tempo real.
* **Filtros Inteligentes:** Gráficos e indicadores que se atualizam conforme o período selecionado (7, 30 ou 90 dias).
* **Responsividade Total:** Interface adaptativa para smartphones, tablets e desktops.

---

## 🛠️ Tecnologias Utilizadas

* **React:** Biblioteca base para a construção dos componentes e estado.
* **Chart.js:** Renderização de gráficos complexos (Bar, Line, Doughnut).
* **CSS3 Custom Properties:** Sistema de temas variáveis para suporte nativo a Dark Mode.
* **Material Symbols:** Biblioteca de ícones arredondados do Google.

---

## ⚙️ Funcionalidades Principais

1.  **Gestão de Pedidos:** Tabela funcional com filtros por status e busca global por cliente ou produto.
2.  **Métricas Detalhadas:** Seção dedicada para análise de conversão, ticket médio e receita acumulada.
3.  **Lista de Clientes:** Gerenciamento de base de usuários com indicadores de status (Ativo/Inativo).
4.  **UX Mobile:** Menu lateral estilo "drawer" (gaveta) e tabelas com scroll horizontal otimizado.

## 🚀 Como Executar o Projeto

Para rodar este projeto localmente, siga os passos abaixo:

```bash
# 1. Clone o repositório
git clone [https://github.com/seu-usuario/liteadmin.git](https://github.com/seu-usuario/liteadmin.git)

# 2. Acesse a pasta do projeto
cd liteadmin

# 3. Instale as dependências
npm install

# 4. Inicie o servidor de desenvolvimento
npm start
